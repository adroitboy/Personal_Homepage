The Estimation of Pulse Transit Time (PTT
(June. 2013 - Mar. 2014)

  • Project Description: This project developed a new system to continuously measure blood pressure based on the principle proposed by Lansdown, that the pulse wave transit time shows a linear correlation with arterial blood pressure within a certain range. We first detect the subjects ECG and pulse wave signal to calculate the PTT (pulse transit time), based on which the blood pressure can be computed efficiently.

Out-of-Order Pipeline Modeling & Analysis in Real-Time Systems
(Apr. 2013 - June. 2013)

    Project Description: For real-time systems, in order to support schedulability analysis, it is crucial to estimate the Worst-Case Execution Time (WCET) of a program. WCET analysis typically models the timing effects of mirco-architectural features in modern processors (such as pipeline, cache, branch prediction) to obtain safe and tight estimates. In this work, we mainly used constraint solving to model the behavior of an out-of-order superscalar pipeline, which can then be used to computer the WCET of a program.
